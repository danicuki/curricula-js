# Fórum de Discussão

- Tipo: `oficina`
- Formato: `individual`
- Duração: `2h`

***

## Enunciado

Criaremos um fórum no qual podemos criar tópicos e responder a esses tópicos.

Um tópico possui:

- Um título;
- Um contador de número de respostas;
- Uma autora.

Uma resposta tem:

- Um *topic_id* (para saber de qual tópico é a resposta);
- O conteúdo da resposta;
- A autora.
